## 二、已确认的总体要求

- 新建全新项目，不直接修改 `Reference`。
- Windows Server + IIS + SQL Server。
- 响应式 Web + PWA，电脑和手机共用一套系统。
- 有限离线仅覆盖阶段成果草稿、工程量临时录入、备注和照片。
- 单公司使用，但支持多个部门、分支机构和我方签约公司。
- 系统级管理员拥有全部系统和业务权限；应用级管理员负责日常业务管理。
- 合作单位统一建档，支持甲方/总包、施工班组、材料供应商、零星供应商多角色。
- 项目可以有多份合同、多个我方签约公司和工程清单。
- 支持暂估量价、结算量价、部分结算和自动金额计算。
- 财务是内部经营台账，不建设完整会计系统。
- 员工分正式员工和劳务员工，工资支持阶段批次、多种工资项目和分批支付。
- 支持简单报销、借支、分红/利息等员工往来。
- 第一阶段不做审批工作流、原生 App、完整甘特图和供应商/班组登录。
- 支持老系统数据平移、主要模块 Excel 导入导出、领导经营汇总台账、数据库和附件备份。
- 支持图表、指标卡、风险提醒、自由筛选字段导出、个人模板和管理员共享模板。
